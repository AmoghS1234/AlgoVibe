Group Chat Clusters: Who's Connected to Whom?

Find Friend Groups using DFS & Water Flow Animation

Input

×

Problem Statement

Students send direct messages forming an undirected graph where edges represent communication between two people.

Some students are part of large interconnected friend groups while others are isolated or in small clusters. Count how many separate friend groups exist by finding connected components.

Technical Challenge

Use DFS or BFS to identify all connected components. Each component represents one friend group. The visualization shows the graph with water-like flow across connected nodes, coloring each cluster differently.

Input Format

n m
u v  (m lines)

Output Format

Number of separate friend groups
×

Sample Test Cases

Input:
6 4
1 2
2 3
4 5
5 6

Output:
2

Explanation: Students {1,2,3} form one group and {4,5,6} form another group.
